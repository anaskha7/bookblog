# Wireframe Estructural (Boceto)

Este documento describe la estructura visual ("Layout") planificada para la página principal.

## Estructura de la Página de Inicio

```text
+---------------------------------------------------------------+
|  HEADER / NAVBAR                                              |
|  [Logo: BookBlog]    [Inicio] [Buscar]       [Login/Registro] |
+---------------------------------------------------------------+
|                                                               |
|  HERO SECTION (Destacado)                                     |
|  +---------------------------------------------------------+  |
|  |  TITULO DEL ARTÍCULO PRINCIPAL                          |  |
|  |  Extracto breve del contenido...                        |  |
|  |  [Leer más]                             [IMG GRANDE]    |  |
|  +---------------------------------------------------------+  |
|                                                               |
+---------------------------------------------------------------+
|  MAIN CONTENT AREA                                            |
|                                                               |
|  +--------------------------+    +-------------------------+  |
|  |  LISTADO DE POSTS (70%)  |    |  SIDEBAR / FILTROS (30%)|  |
|  |                          |    |                         |  |
|  |  [Card Post 1]           |    |  [Buscar.........]      |  |
|  |  + Img                   |    |                         |  |
|  |  + Titulo                |    |  Categorías:            |  |
|  |  + Autor / Fecha         |    |  - Ficción              |  |
|  |                          |    |  - Ensayo               |  |
|  |  [Card Post 2]           |    |  - Poesía               |  |
|  |  + Img...                |    |                         |  |
|  |                          |    |  Etiquetas:             |  |
|  |  [Card Post 3]           |    |  [Novedad] [Clásico]    |  |
|  |  ...                     |    |                         |  |
|  +--------------------------+    +-------------------------+  |
|                                                               |
+---------------------------------------------------------------+
|  FOOTER                                                       |
|  [Links Legales]    [Redes Sociales]    [Copyright 2024]      |
+---------------------------------------------------------------+
```

## Elementos Clave
1.  **Navegación Fija**: Siempre visible para facilitar el acceso.
2.  **Hero Section**: Un área visualmente impactante para atrapar la atención del lector inmediatamente.
3.  **Grid Asimétrico**: El contenido principal (posts) ocupa la mayor parte del ancho, apoyado por una barra lateral para navegación secundaria y descubrimiento de contenido.
