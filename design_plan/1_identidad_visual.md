# Identidad Visual y Estilo - Book Project

## Concepto
El proyecto se centra en un "Blog de Curaduría Literaria". El objetivo es transmitir una sensación de calma, elegancia y profundidad, similar a leer un libro físico de alta calidad o visitar una librería clásica pero moderna.

## Paleta de Colores
Se ha seleccionado una paleta cálida y "editorial" para evitar la fatiga visual y evocar el papel impreso.

| Color | Hex | Uso |
|-------|-----|-----|
| **Warm Cream** | `#FFFBF5` | Fondo principal. Suave, imita el papel crema. |
| **Warm Black** | `#1C1917` | Texto principal. Menos agresivo que el negro puro. |
| **Sienna Brown** | `#A0522D` | Color primario. Botones, enlaces, acentos, identidad. |
| **Amber** | `#F59E0B` | Detalles secundarios, valoraciones (estrellas). |
| **Stone Gray** | `#78716c` | Texto secundario y bordes sutiles. |
| **Pure White** | `#FFFFFF` | Fondo de tarjetas para resaltar sobre el crema. |

## Tipografía
La combinación tipográfica busca el equilibrio entre la legibilidad moderna y el carácter literario clásico.

*   **Títulos (Serif):** `Playfair Display`. Elegante, con alto contraste. Se usa en encabezados y citas.
*   **Cuerpo (Sans-serif):** `Inter`. Limpia, moderna y altamente legible para bloques de texto largos y la interfaz general.

## Estilo General
*   **Diseño:** "Glassmorphism" sutil y bordes redondeados amplios para suavizar la interfaz.
*   **Textura:** Uso de patrones de ruido o tramas sutiles de fondo para dar textura de papel.
*   **Interacciones:** Transiciones suaves (fadeIn, hover states) para mantener la sensación de calma.
